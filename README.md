## `whoami`

**Senior** mobile engineer from India, currently at **[GeekyAnts](https://www.geekyants.com/)**, where I spend most days shipping production Flutter apps and the NestJS services that back them.

I have a soft spot for **interaction design** — the kind of UI that responds before you finish the gesture. My pub.dev packages live in that space.

Deeper in the stack, I've spent a lot of time in the gnarly bits — **Firebase Cloud Messaging**, **Cloud Functions**, and **deep linking** — the parts that work great in the demo and break in three different ways in production. Occasional **React** on the side when a project asks for it.

Off-hours I write at **[vikaskumar75.hashnode.dev](https://vikaskumar75.hashnode.dev/)** about the parts of mobile and telecom that don't get enough light.
